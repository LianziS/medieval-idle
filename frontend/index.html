🎮 游戏简介

《中世纪雇佣兵》是一款放置类养成游戏。你将扮演一名中世纪的雇佣兵，通过伐木、挖矿、采集、制作等行动获取资源，升级技能，打造装备，逐步成长为一名强大的冒险者。

🪓 技能系统

游戏共有 8 个技能，每个技能都有独立的等级和经验值：

  • 伐木 - 砍伐各种树木获取木材，从青杉到世界树
  • 挖矿 - 开采矿石，从青闪矿到月华结晶
  • 采集 - 在野外采集草药、果实、羽毛等材料
  • 制作 - 将木材加工成木板，用于建造和锻造
  • 锻造 - 将矿石冶炼成矿锭，打造各种工具
  • 缝制 - 将采集材料加工成布料和装备
  • 炼金 - 制作各种药剂
  • 酿造 - 酿造各种酒类

🏠 建筑系统

建造和升级建筑可以解锁更多功能：

  • 帐篷 - 核心建筑，升级可解锁其他建筑
  • 伐木场 - 解锁更高级的树木
  • 矿洞 - 解锁更高级的矿石
  • 锻造屋 - 打造工具
  • 木工坊/裁缝铺/炼金小屋/酿酒坊 - 解锁相应制作

⚔️ 装备系统

在「我的物品 → 装备栏」中可以装备工具。每个技能都有对应的装备槽位：

  • 装备工具可以获得 加速加成，提高行动效率
  • 工具需要在锻造屋打造，消耗矿锭和木板
  • 高级工具需要更高的锻造等级和相应技能等级

📊 升级经验公式

技能升级所需经验：100 × 1.5^(等级-1)

💡 新手建议

  1. 先从伐木和挖矿开始，收集基础资源
  2. 升级帐篷解锁更多建筑
  3. 在锻造屋打造青闪斧和青闪镐，提高效率
  4. 资源等级越高，经验越多，但需要更高的技能等级
  5. 点击行动卡片的 ∞ 按钮 可以无限循环执行
  6. 可以在行动进行时将新行动加入队列

🔥 资源等级一览

木材：青杉(Lv.1) → 铁桦(Lv.10) → 风啸树(Lv.20) → 焰心树(Lv.35) → 霜叶枫(Lv.50) → 雷鸣树(Lv.65) → 古橡(Lv.80) → 世界树(Lv.95)

矿石：青闪矿(Lv.1) → 赤铁矿(Lv.10) → 羽石矿(Lv.20) → 白鸠矿(Lv.35) → 狱炎矿(Lv.50) → 雷鸣矿(Lv.65) → 曜光结晶(Lv.80) → 月华结晶(Lv.95)

工具：青闪系列(15%加成) → 赤铁系列(22.5%) → 羽系列(30%) → 白银系列(45%) → 狱炎系列(60%) → 雷鸣系列(75%) → 璀璨系列(90%) → 星辉系列(105%)
