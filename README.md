# 中世纪雇佣兵 - 放置游戏

一款中世纪风格的放置类网页游戏，玩家扮演雇佣兵，通过伐木、挖矿、采集、锻造、缝制等技能积累资源，打造装备，与商人交易，逐步成长。

## 核心玩法

### 🎮 技能系统

| 技能 | 说明 | 产出 |
|------|------|------|
| 🪓 伐木 | 砍伐各种树木 | 木材 → 木板 |
| ⛏️ 挖矿 | 开采矿石 | 矿石 → 矿锭 |
| 🌾 采集 | 采集草药、纤维等 | 草药、布料原料 |
| 🔨 制作 | 将木材加工成木板 | 木板 |
| ⚒️ 锻造 | 冶炼矿锭、打造工具 | 矿锭、斧/镐/凿/针/镰 |
| 🧵 缝制 | 制作各种布料 | 布料 |

### 🛠️ 装备系统

- 5 个装备槽：斧头、镐子、凿子、针、镰刀
- 每种工具 8 个等级（青闪 → 星辉）
- 装备提供速度加成，提高采集效率

### 🏪 商人系统

- 4 位商人：建筑师、铸甲师、缝缀师、药剂师
- 好感度系统：通过任务提升好感度
- 任务系统：提交资源获取奖励

### 📋 行动队列

- 支持批量行动（1/10/100/1000/∞ 次）
- 最多 4 个行动排队等待
- 随时可取消当前行动

## 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **存储**: LocalStorage 自动存档
- **部署**: GitHub Pages

## 项目结构

