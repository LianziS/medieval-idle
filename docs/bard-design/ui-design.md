# 吟游诗人系统 — UI 设计说明文档

## 二、技能头部（顶部固定区域）

位于页面顶部，圆角卡片样式，包含以下内容：

| 元素 | 说明 |
|---|---|
| 标题 | 左侧显示"🎵 吟游诗人" |
| 状态徽章 | 右侧显示当前诗人状态（空闲 / 出游中 / 演奏中），带绿色圆点动画指示 |
| 等级 | 右侧显示当前等级，如"Lv. 8" |
| 经验条 | 红色渐变进度条，显示当前经验占比 |
| 经验详情 | 进度条下方小字，格式："当前经验 / 升级经验 · 出游×N次 · 演奏×N次" |

头部下方有一条棕色分隔线，与角色卡分隔。

## 三、角色信息卡

紧接技能头部下方，横向布局，分为左右两部分：

### 左侧：头像
100×100 像素方块，暗红边框，中央显示大号音符图标。

### 右侧：四项属性网格
响应式网格排列，每项为小圆角卡片，包含图标 + 标签 + 数值：

| 属性 | 图标 | 示例 |
|---|---|---|
| 出游时长 | ⏱️ | 8小时00分（受鞋子等级影响） |
| 笔加成 | ✨ | 精良+15% / 史诗+10%（受笔等级影响） |
| 已解锁 | 🗺️ | 4 / 4 目的地 |
| 乐谱总数 | 📜 | 12 张 |

## 五、Tab 1：出游派遣

### 5.1 目的地选择网格

标题"🗺️ 选择目的地"，下方为四个等宽卡片，横向排列：

| 卡片 | 图标 | 等级 | 说明 |
|---|:---:|:---:|---|
| 低语湾 | 🌊 | Lv.1 | 狼林边缘 |
| 河间谷地 | 🏞️ | Lv.3 | 河间地带 |
| 峡谷营地 | ⛰️ | Lv.6 | 叹息峡谷 |
| 黑石灯塔 | 🗼 | Lv.8 | 龙脊山脉 |

**交互：**
- 单击：弹出目的地详情弹框（查看品质概率、掉落物、经验等信息）
- 弹框底部有"确定"按钮，点击后选中该目的地（红色边框高亮 + 微红背景），下方消耗品区自动刷新数值

### 5.2 消耗品与派遣区

目的地网格下方的暗色圆角容器，纵向排列以下内容：

**金币消耗**
- 格式：💰 金币 [拥有量 / 需求量]
- 拥有量足够时数字显示绿色，不足时显示红色

**手稿消耗**
- 格式：📜 手稿 [拥有量 / 需求量]
- 颜色规则同金币

**酒箱选择**
- 标签"🍷 酒箱"，右侧为一个 48×48 像素正方形图标框
- 框内显示当前选中酒箱的图标（🍺/🍷/🥃），右下角显示数量（如 ×12）
- 点击：弹出酒箱选择弹框
- 悬浮：显示 tooltip（名称、经验增益百分比、库存数量）

**预览栏**
- 暖色背景条，显示两项预估信息：
  - 预计时长：如"8h"（受鞋子等级缩减）
  - 经验：如"50 exp"，若使用了中级/高级酒箱则附加 (+20%) / (+40%) 标签

**派遣按钮**
- 全宽红色渐变按钮，默认显示"🚀 出发前往XX"
- 所有材料（金币 + 手稿 + 酒箱）充足时可点击
- 任一材料不足时按钮变灰，显示"⚠️ 材料不足"，不可点击

## 六、Tab 2：演奏

演奏区整体最大宽度 360px，纵向布局：

### 6.1 乐谱选择槽

标题"选择"（灰色小字），下方为一个 56×56 像素正方形框：

- **未选择状态**：虚线边框，灰色半透明，中央显示灰色文字"选择"
- **已选择状态**：实线边框，显示乐谱图标 + 右下角库存数量
- **悬浮**：显示 tooltip（乐谱名称 + 品质徽章、可用于、效果、演奏时长、库存）
- **点击**：弹出乐谱选择弹框

### 6.2 乐谱详情

选中乐谱后，选择框下方显示六行详情信息：

| 行 | 标签 | 示例值 |
|---|---|---|
| 乐谱 | 品质徽章 | 大地之艺 精良 |
| 演奏时长 | 分钟数 | 30分钟 |
| 可用于 | 适用职业 | 伐木 / 挖矿 / 采集 |
| 效果 | 增益描述 | 10% 加工转化（橙色加粗） |
| 详情 | 效果详细解释 | 有几率立即将原材料转化成加工品… |
| 获得经验 | 分隔线下方 | 15 exp（绿色加粗） |

### 6.3 进度条

演奏进行中显示：

- 蓝色渐变进度条（10px 高）
- 右侧百分比文字（如"42%"）
- 完成后显示"✅ 演奏完成"，随后自动恢复到未选择状态

### 6.4 演奏按钮

- 未选择乐谱时：灰色禁用状态，显示"🎵 开始演奏"
- 已选择时：红色渐变可点击，显示"🎵 开始演奏"
- 演奏中：显示"🎵 演奏中..."，禁用

点击后：消耗 1 张所选乐谱，开始倒计时，完成后恢复默认状态。

## 七、Tab 3：诗人装备

### 7.1 装备槽位网格

标题"🎒 诗人装备"，下方为三个装备卡片（笔、鞋子、乐器），等宽横向排列：

每个槽位有三种状态：

| 状态 | 外观 | 说明 |
|---|---|---|
| 已装备 | 实线暗红边框，显示装备图标 + 名称 + 等级 | 悬浮显示 tooltip（名称、等级、当前增益数值） |
| 未装备 | 虚线边框，半透明，显示槽位图标 + "未装备" | 背包中有该类型但未穿上 |
| 未获得 | 虚线边框，半透明，图标更暗，显示"未获得" | 背包中完全没有该类型装备 |

点击任意槽位：弹出装备选择弹框。

## 八、Tab 4：成长记录

### 8.1 统计汇总

复用角色卡的横向布局样式（无头像），四项统计数据：

| 属性 | 示例 |
|---|---|
| 出游次数 | 42 次 |
| 演奏次数 | 7 次 |
| 获得乐谱总数 | 47 张 |
| 获得史诗乐谱 | 3 张 |

### 8.2 等级解锁表

标题"🔓 等级解锁内容（20级）"，下方为 20 张等级卡片的响应式网格。

每张卡片从上到下包含：

| 元素 | 说明 |
|---|---|
| 等级 | "Lv.1" ~ "Lv.20"，等级数字棕色加粗 |
| 解锁内容 | 该等级解锁的功能描述 |
| 经验需求 | 升到下一级所需经验 |
| 状态标签 | ✓ 已解锁（绿色） / 🔒 2,431 / 3,000（橙色，下一级） / 🔒（灰色，未解锁） |

三种视觉状态：
- **已解锁**：绿色边框 + 绿色微光背景
- **下一级（即将解锁）**：橙色边框 + 橙色发光
- **未解锁**：默认暗色边框

## 九、弹框系统

所有弹框共用同一个模态层，屏幕居中显示，最大宽度 340px（装备选择 440px）。

### 通用结构
- 半透明黑色遮罩层覆盖全屏
- 弹框容器：深色背景 + 棕色边框 + 圆角
- 右上角关闭按钮（✕）
- 弹框底部内容过多时可纵向滚动
- 点击遮罩层或按 ESC 关闭

### 9.1 目的地详情弹框（点击目的地卡片触发）

| 区域 | 内容 |
|---|---|
| 头部 | 目的地图标 + 名称 + 地区 · 解锁等级 |
| 描述 | 一段文字描述 |
| 乐谱品质概率 | 当前笔等级下的品质分布：普通（绿色徽章）X% / 精良（蓝色徽章）X% / 史诗（紫色徽章）X% |
| 掉落物 | 每行一个物品：数量 + 物品名称标签 + 掉落概率 |
| 经验 | 分隔线下方显示"获得经验：50 exp" |
| 确定按钮 | 底部全宽按钮，点击后选中该目的地并关闭弹框 |

### 9.2 酒箱选择弹框（点击酒箱图标触发）

- 标题"选择酒箱"
- 三列网格，每格为一个酒箱选项：
  - 中央大图标 + 右下角库存数量
  - 当前选中的显示红色边框
- 悬浮显示 tooltip（名称、经验增益、库存）
- 点击：选中该酒箱，关闭弹框，派遣面板自动刷新

### 9.3 乐谱选择弹框（点击演奏选择框触发）

- 标题"选择乐谱"
- 自适应换行网格，共 9 格（3 类 × 3 品质）：
  - 每格 56×56 像素，显示乐谱类别图标 + 右下角库存数量
  - 当前选中的显示红色边框
- 悬浮显示 tooltip（名称 + 品质徽章、可用于、效果、演奏时长、库存）
- 点击：选中该乐谱，关闭弹框，演奏面板显示乐谱详情

### 9.4 装备选择弹框（点击装备槽位触发）

- 标题"选择笔/鞋子/乐器"
- 三列网格，每格为一个装备选项：
  - 装备图标 + 名称 + 等级 + 右下角"已装备"或"×N"
  - 当前已装备的显示红色边框 + 微红背景
- 悬浮显示 tooltip（名称、等级、增益数值）
- 点击：穿戴该装备，关闭弹框，装备栏刷新
- 底部"卸下当前装备"按钮（仅在有装备时显示）：点击卸下当前装备

## 十一、Tooltip（悬浮提示）

页面中多处使用 tooltip 显示详细信息，统一规则：

- 默认显示在触发元素上方居中
- 上方空间不足时显示在下方
- 下方也不足时显示在右侧
- 使用 position: fixed 定位，不受弹框滚动裁剪
- 深色背景 + 棕色边框 + 投影，鼠标移开后自动隐藏

适用位置：装备槽位、酒箱图标、乐谱图标、弹框内的酒箱选项、弹框内的乐谱选项、弹框内的装备选项。
